Some Previous Projects

Automated Utilities Billing for Rental Properties
After installing smart meters in to several apartments that read utilities usage and uploaded the data to a web server, I built a system in Java that connects via FTP to the server, downloads the data, then uploads it to a MySQL database hosted on AWS. I also built a separate GUI using Java Swing to allow users to choose building, tenant, and dates, and then the system automates the process of creating the utilities bill and sending it to the tenant.
Twitter Sentiment Analysis
I used the Python Tweepy library to access live Twitter data that could be filtered based on a variety of factors. I created a sentiment analysis algorithm based on the bag of words model and a large set of pre-categorized training data to classify Tweets as being positive or negative. I also created an interface that plots the results.
Pacman Deep Learning Algorithms
This was a group project for my Deep Learning class in which our team (consisting of 2 other members) implemented several deep reinforcement learning algorithms in order to find which was the most effective at winning Pacman games. We used a library from online that had already built out an agent using Deep-Q-Networks as well as the Pacman game logistics, and further developed agents that implemented the DDQN, DDDQN, and DDPG algorithms. We also designed a grid search to find and tweak the optimal hyperparameters. All of this was done using the PyTorch Library.
